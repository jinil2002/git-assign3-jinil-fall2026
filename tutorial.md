# Git Tutorial Learning 

For this assignment, I explored Git tutorials such as Git Immersion and Learn Git Branching.

## What I Learned

### 1. Git Basics
Git is a version control system that helps track changes in files and allows multiple people to collaborate on projects.

### 2. Repository
A repository (repo) is a folder that contains project files and tracks their changes.

### 3. Important Commands
- `git init` – Initialize a new repository
- `git add` – Add files to staging area
- `git commit` – Save changes with a message
- `git push` – Upload changes to GitHub
- `git pull` – Get latest changes

### 4. Branching
Branches allow developers to work on different features without affecting the main project.

### 5. GitHub
GitHub is a platform to store and share Git repositories online.
